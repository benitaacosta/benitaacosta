Invitations

These stunning die-cut invitations are a combination of an inner print enclosed by gate style folds, created in craft cutting software.

These invitations were printed on a wide format printer with registration marks that were then sent to a craft cutter for accurate cutting. In Photoshop, I created several color options of these masks by using custom glitter swatches and adjusting color hues.

Using the same mask file for the cut invitations and using various font styles and custom brushes, I created this client requested standard print version for mass printing.

Included with this package were RSVP cards I designed to match the style and color scheme of the invitations. This digital sample is ideal for presenting clients with an idea of their finished product(s).

These graduation invitations are a print and die-cut combination. I created the layout using photo effects, masks, and font styles, to be accurately sent for print and cut.

These papel picado inspired invitations are great for fiesta or Coco themed parties. With an editable gradient background, fiesta font styles, and die-cut shapes.

These Beauty and the Beast inspired cut-out hand mirror invitations are ideal for princess birthday parties. I created the mirror and accents and set up the layout using font styles and images in preparation for print and cut.

These are WWE inspired pass style invitations. I created the name banner to resemble the Wrestlemania title, as well as set up the layout using font styles and created the generic barcode. Laminated and attached to a lanyard to complete the pass style.

This Powerpuff Girls inspired invitation is a combination of a standard print and cut out. I photographed the child, isolated the image for a transparent background, and created the layout including various shapes and font styles. A cut out of the child was inserted through a slit made in the print to give the interactive illusion that the child is flying.
